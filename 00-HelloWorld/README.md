# Trabajo Nº 00 - Hello World

## Análisis del Problema

### Transcripción del problema
Adquirir y preparar los recursos necesarios para resolver los trabajos del curso. Se debe instalar y probar un compilador de C, creando un programa simple que imprima "Hello, World!" y verificando su ejecución.

Se espera que el programa compile correctamente y muestre por pantalla el mensaje "Hello, World!". Además, se espera que la salida pueda redirigirse a un archivo y que su contenido sea correcto.

### Modelo IPO

**Entradas:**
* No hay entradas por parte del usuario.

**Proceso:**
* Ejecutar un programa que imprime un mensaje en consola.

**Salidas:**
* Mensaje "Hola mundo!" (tipo: string)
* Archivo `output.txt` con el mismo contenido

---

## Diseño de la Solución

### Léxico del Algoritmo
* **printf:** función utilizada para imprimir en pantalla.
* **main:** función principal del programa.
* **output.txt:** archivo donde se redirige la salida.

### Algoritmo (representación textual)
1. Iniciar programa.
2. Imprimir el mensaje "Hola mundo!".
3. Finalizar programa.
